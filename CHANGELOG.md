# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2024-11-14

### Added
- Initial project structure
- Pokemon Showdown WebSocket client
- Message parser for battle protocol
- Battle state management system
- Data models (Pokemon, Moves, Types, etc.)
- Type effectiveness calculator
- Damage calculator with Gen 9 formula
- Random agent baseline
- Heuristic agent with rule-based decisions
- Battle handler for coordinating battles
- Logging and configuration utilities
- Main entry point with CLI
- Test suite for core components
- Comprehensive documentation
- Setup scripts

### Features
- Connect to Pokemon Showdown servers
- Authenticate with username/password or as guest
- Parse battle messages in real-time
- Track complete battle state
- Make intelligent decisions based on heuristics
- Calculate damage ranges accurately
- Handle type effectiveness (including immunities)
- Support for multiple battle formats
- Extensible agent framework
